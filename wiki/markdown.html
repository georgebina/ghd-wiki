Markdown
Edit online

Markdown can also be used to create pages, so people with Markdown only knowledge will still be able to create a page and use Markdown as the source for that.

For now the editing of Markdown content is directed to the Github built-in editors. In the future we plan to support Markdown editing similar to the one already available in the oXygen XML Editor also in the oXygen XML Web Author thus providing

  • syntax highlight
  • specific Markdown smart actions and toolbar
  • live DITA preview
