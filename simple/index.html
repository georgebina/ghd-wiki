DITA 2 Wiki frank

This project publishes DITA content from the 'wiki' folder automatically using an XSLT script that is triggered on each commit on the master branch through Travis. The DITA content is converted to HTML and commited on the 'gh-pages' branch which serves the content through GitHub Pages.

GitHub is used as content storage and the oXygen XML Web Author is used for online editing.
